# Catalyst Path Template

## Purpose
State the path that will validate or break the thesis, using a dated catalyst for tactical ideas or an observable milestone for compounders.

## Required Fields
- <<REQUIRED>> Stock / setup:
- <<REQUIRED>> Mode {tactical catalyst / compounder milestone}:
- <<REQUIRED>> Named catalyst or milestone:
- <<REQUIRED>> Target date or observation window:
- <<REQUIRED>> Observable proof point:
- <<REQUIRED>> Failure mode if the path slips:

## Guidance
- Tactical mode should name an event with a date or narrow window.
- Compounder mode should still have an observable milestone even if exact timing is flexible.
- Tie the proof point to a public data release where possible.

## Example
```md
Stock / setup: 2603 into contract-rate resets.
Mode {tactical catalyst / compounder milestone}: tactical catalyst
Named catalyst or milestone: 2026 Q3 contract reset commentary.
Target date or observation window: July to August 2026.
Observable proof point: Freight-rate renewal trends and monthly revenue stabilization.
Failure mode if the path slips: Spot rates roll over before contracted rates are reset.
```
