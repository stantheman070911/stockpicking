# Red Flag Screen Template

## Purpose
Summarize the fast governance and forensic checks that determine whether deeper workstream-B review is required.

## Required Fields
- <<REQUIRED>> Stock / setup:
- <<REQUIRED>> Auditor change check:
- <<REQUIRED>> Governance keyword hits:
- <<REQUIRED>> CFO/NI flag:
- <<REQUIRED>> Debt wall proxy:
- <<REQUIRED>> Dilution history:
- <<REQUIRED>> Manual follow-up required:

## Guidance
- This template is for the stage-one screen, not the full deep forensic memo.
- Mark unclear items as manual review or not assessed rather than silently passing them.
- Keep each line tied to a source or dataset when the code later populates it.

## Example
```md
Stock / setup: 2609 after freight-rate rebound.
Auditor change check: No recent hit in the news scan.
Governance keyword hits: None in the current lookback window.
CFO/NI flag: Yellow flag; cash conversion is weak but not a hard fail.
Debt wall proxy: Short-term debt mix elevated; manual maturity review required.
Dilution history: No recent capital reduction or repeat dilution pattern detected.
Manual follow-up required: Yes, because the debt wall proxy triggered.
```
