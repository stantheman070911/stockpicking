# Convertible Bond Manual Review Template

## Purpose
Preserve a place for convertible-bond diligence when the default free-tier path cannot automate those checks.

## Required Fields
- <<REQUIRED>> Stock / setup:
- <<REQUIRED>> Outstanding CB confirmed:
- <<REQUIRED>> Conversion price:
- <<REQUIRED>> Put / call dates:
- <<REQUIRED>> Observed equity dilution risk:
- <<REQUIRED>> Manual source log:

## Guidance
- Use this only when a name actually has a convertible-bond overlay worth tracking.
- Distinguish between verified public-disclosure facts and inference.
- If no CB is outstanding, say so plainly and close the section.

## Example
```md
Stock / setup: Mid-cap tech name with prior CB issuance.
Outstanding CB confirmed: Yes, via public disclosure review.
Conversion price: NT$125.
Put / call dates: Investor put in September 2027; issuer call after conversion threshold is met.
Observed equity dilution risk: Moderate if the equity rerates into the conversion band.
Manual source log: TWSE filing, company annual report, and CB term sheet.
```
